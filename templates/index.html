Hot keys

To make it so that you can open the program when you press a key or when you press a combination of keys, you must enter the key to be pressed and the full path to the file to be opened.

To add key combinations, you need to type key + key, example: s + d.

Attention more than 1 + is impossible and it is impossible to drive the upper register of keys.
